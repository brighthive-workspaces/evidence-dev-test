```sql student_attendance
SELECT
    student_name,
    attendance_rate
FROM
    student_attendance_data
ORDER BY
    student_name
```

```jsx
<BarChart
    data={student_attendance}
    x=student_name
    y=attendance_rate
    labels=true
    height=300
    title="Student Attendance Rates"
    color="#2c7fb8"
/>
```
