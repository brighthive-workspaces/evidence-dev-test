# Student Attendance Rates Visualization

## SQL Query
```sql student_attendance
SELECT 
    student_id,
    student_name,
    attendance_rate
FROM 
    student_attendance_data
ORDER BY 
    attendance_rate DESC;
```

## Markdown for Visualization

<BarChart
    data={student_attendance}
    x=student_name
    y=attendance_rate
    labels=true
    color="#4e79a7"
    height=400
    title="Student Attendance Rates"
    swapXY=true
/>
