```sql student_attendance
SELECT
    student_name,
    attendance_rate
FROM
    student_attendance
ORDER BY
    attendance_rate DESC
```

```jsx
<BarChart
    data={student_attendance}
    x=student_name
    y=attendance_rate
    labels=true
    type=grouped
    height=300
    yFmt="percent"
    colorPalette={["#4CAF50"]}
/>
```
