```sql students_attendance
SELECT
    student_name,
    attendance_rate
FROM
    student_attendance
ORDER BY attendance_rate DESC
```

```jsx
<BarChart
    data={students_attendance}
    x=student_name
    y=attendance_rate
    labels=true
    colorPalette={["#0072b2", "#d55e00"]}
    height=400
/>
```
