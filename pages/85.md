### SQL Query
```sql top_students
SELECT s.first_name,
       s.last_name
FROM "workspace-database"."brighthive_dev_redshift"."students" s
JOIN "workspace-database"."brighthive_dev_redshift"."academic_performance" ap
  ON s.student_id = ap.student_id
WHERE ap.final_grade = 'A';
```
