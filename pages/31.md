```sql students_attendance
SELECT 
    student_name,
    attendance_rate
FROM 
    student_attendance
ORDER BY 
    attendance_rate DESC
```

```jsx
<BarChart
    data={students_attendance}
    x=student_name
    y=attendance_rate
    labels=true
    title="Student Attendance Rates"
    color="#4CAF50"
    xFmt="string"
    yFmt="percent"
    height=400
/>
```
