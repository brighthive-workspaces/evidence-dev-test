### SQL Query
```sql student_achievers
SELECT 
    s.major, 
    COUNT(s.first_name) as student_count
FROM 
    brighthive_dev_redshift.students s 
JOIN 
    brighthive_dev_redshift.academic_performance a 
    ON s.student_id = a.student_id 
WHERE 
    a.final_grade = 'A'
GROUP BY 
    s.major;
```
