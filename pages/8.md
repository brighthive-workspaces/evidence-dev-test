### SQL Query to Calculate Student Attendance Rates

```sql student_attendance
SELECT
    student_name,
    attendance_rate
FROM
    student_attendance
ORDER BY
    attendance_rate DESC;
```

### Markdown for Bar Chart Visualization

```jsx
<BarChart
    data={student_attendance}
    x=student_name
    y=attendance_rate
    type=grouped
    xFmt="none"
    yFmt=pct1
    labels=true
    colorPalette={["#4CAF50", "#FF9800", "#F44336"]}
/>
```
