### SQL Query
```sql attendance_by_student
SELECT 
    s.first_name, 
    a.attendance_rate
FROM 
    database_891377033216.academic_performance a 
JOIN 
    database_891377033216.students s 
ON 
    a.student_id = s.student_id
WHERE 
    a.final_grade = 'A' 
    AND a.semester = 'Fall 2023'
ORDER BY 
    s.first_name;
```
