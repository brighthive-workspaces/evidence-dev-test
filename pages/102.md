### SQL Query
```sql student_gpa_by_major
SELECT 
    s.major,
    SUM(CAST(ap.gpa AS FLOAT)) / COUNT(*) AS avg_gpa
FROM 
    brighthive_dev_redshift.students s
JOIN 
    brighthive_dev_redshift.academic_performance ap
    ON s.student_id = ap.student_id
GROUP BY 
    s.major
ORDER BY 
    avg_gpa DESC;
```
