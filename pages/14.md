### SQL Query for Student Attendance Rates
```sql student_attendance
SELECT
    student_name,
    attendance_rate
FROM
    student_attendance_records
ORDER BY
    student_name
```

### Evidence.dev Bar Chart Component
```jsx
<BarChart
    data={student_attendance}
    x=student_name
    y=attendance_rate
    labels=true
    colors={["#4B0082"]}
    height=400
    xLabel="Student Name"
    yLabel="Attendance Rate"
/>
```
