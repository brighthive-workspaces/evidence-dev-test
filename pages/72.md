### SQL Query
```sql student_achievements
SELECT 
    s.first_name, 
    COUNT(*) AS number_of_students
FROM 
    brighthive_dev_redshift.academic_performance ap
JOIN 
    brighthive_dev_redshift.students s 
ON 
    ap.student_id = s.student_id
WHERE 
    ap.final_grade = 'A'
GROUP BY 
    s.first_name
ORDER BY 
    number_of_students DESC;
```
