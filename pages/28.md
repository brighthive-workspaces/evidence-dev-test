```sql student_attendance
SELECT 
    student_name, 
    attendances::float/total_classes as attendance_rate
FROM 
    student_attendance_records
ORDER BY 
    attendance_rate DESC
```

```jsx
<BarChart
    data={student_attendance}
    x=student_name
    y=attendance_rate
    labels=true
    yFmt="pct2"
    title="Student Attendance Rates"
    colors={['#0073e6']}
    height=400
/>
```
