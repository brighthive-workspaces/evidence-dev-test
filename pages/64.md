### SQL Query
```sql grade_distribution
select 
    grade,
    count(*) as count
from brighthive_dev_redshift.academic_performance
where grade in ('A', 'B', 'F', 'IP')
group by grade
order by grade
```

### Visualization Component
<BarChart
    data={grade_distribution}
    x=grade
    y=count
