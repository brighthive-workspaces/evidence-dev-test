```sql student_attendance
SELECT
    student_name,
    attendance_rate
FROM
    student_attendance_records
WHERE
    attendance_rate IS NOT NULL
ORDER BY
    attendance_rate DESC
```

```jsx
<BarChart
    data={student_attendance}
    x=student_name
    y=attendance_rate
    labels=true
    height=400
    colorPalette={['#4CAF50', '#2196F3', '#FF9800', '#F44336']}
    title="Student Attendance Rates"
/>
```
