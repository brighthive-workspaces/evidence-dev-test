### SQL Query
```sql entry_year_by_major
SELECT 
    major,
    entry_year,
    COUNT(student_id) as student_count
FROM 
    "workspace-database"."brighthive_dev_redshift"."students"
GROUP BY 
    major, entry_year
ORDER BY 
    major, entry_year;
```
