## SQL Query for Attendance Rates by First Name

```sql students_attendance
SELECT 
    s.first_name, 
    a.attendance_rate 
FROM 
    brighthive_dev_redshift.students AS s 
JOIN 
    brighthive_dev_redshift.academic_performance AS a 
ON 
    s.student_id = a.student_id 
WHERE 
    a.final_grade = 'A';
```
